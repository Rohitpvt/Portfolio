## 🚀 Overview

This portfolio is a showcase of modern frontend engineering and creative UI/UX design. Optimized for performance and high-fidelity interactions, it serves as a digital identity for my journey in **AI, Web, and Mobile development**.

The project is built on a **100% Data-Driven Architecture**, where all content (Projects, Certificates, Resume, and Skills) is managed via decoupled JSON files, allowing for effortless updates without touching the source code.

---

## ✨ High-Fidelity Features

### 🎨 Visual & Aesthetic Design
- **Glassmorphism 2.0**: Sophisticated frosted-glass UI with multi-layered depth and cyan-purple ambient glows.
- **Section-Aware Backgrounds**: The background ambient color-scheme dynamically shifts as you navigate through different pages.
- **Advanced Typography**: A curated hierarchy using *Outfit* and *Inter* fonts, optimized for legibility and premium feel.
- **Dynamic Scrollbar**: Custom gradient-thumb scrollbar that reacts to page movement.

### ⚡ Interactive UX
- **Interactive 3D Tilt**: High-performance physics-based tilt effects on all project and content cards.
- **Parallax Avatar Depth**: A persistent 3D parallax effect on the profile avatar that reacts to mouse movement.
- **Staggered Sections**: Fluid cross-fade and staggered reveal animations for page transitions.
- **Skill "Power Up"**: Progress bars that initiate an animated growth sequence upon visiting the resume section.
- **Interactive Resume Modals**: Detailed experience bullet points revealed through high-fidelity glassmorphic popups.

### 🧠 Intelligent Backend & Feedback
- **Typewriter Role Cycle**: An animated role cycling title (AI Assistant, Web Designer, etc.) with blinking cursor.
- **Skeleton Loading States**: Seamless shimmering placeholders to eliminate layout shift during data fetching.
- **Neural Particle Network**: An interactive HTML5 Canvas background that logicially connects nodes based on proximity.
- **AJAX Form Submission**: A custom Formspree integration with background processing and localized success notifications.

---

## 🛠️ Tech Stack

- **Core**: HTML5, CSS3 (Vanilla), JavaScript (ES6+).
- **Architecture**: JSON-decoupled data layers.
- **Icons**: IonIcons.
- **Animation Engine**: CSS Keyframes & JavaScript Physics-based math.
- **Form Handling**: Formspree (Integrated via AJAX).
